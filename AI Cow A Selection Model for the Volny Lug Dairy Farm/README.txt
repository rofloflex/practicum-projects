Линейные модели в машинном обучении
Прогнозирование удоя и вкуса молока для расширения фермы

Описание проекта
Вы работаете в IT-компании, которая разрабатывает решения с применением машинного обучения. К вам обратился фермер из молочного хозяйства «Вольный луг», 
который хочет расширить своё стадо. Он заключил контракт с ассоциацией «ЭкоФерма», где сможет тщательно отбирать коров по следующим критериям:
Удой не менее 6000 кг в год
Молоко должно быть вкусным по его оценке
Ваша задача — построить две модели:
Регрессионную модель для прогнозирования годового удоя
Классификационную модель для определения, будет ли молоко вкусным

Описание данных
В распоряжении три CSV-файла:
ferma_main.csv: данные о текущем стаде фермера
ferma_dad.csv: содержит информацию об отцах коров
cow_buy.csv: данные о коровах, которых предлагает «ЭкоФерма»

Признаки охватывают:
Характеристики коровы: порода, возраст, родословная
Корм: питательность, содержание протеина, соотношение сахара к протеину
Молоко: удой, жирность, белок, вкус
Тип пастбища

Этапы работы
1. Загрузка и изучение данных
Импорт данных и проверка структуры
Проверка типов данных и переименование столбцов при необходимости

2. Предобработка
Обработка пропусков и дубликатов
Кодирование категориальных признаков и масштабирование числовых

3. Исследовательский анализ
Статистический обзор признаков
Визуализация (гистограммы, boxplot, scatterplot)
Выводы об аномалиях, выбросах и необходимости преобразований

4. Корреляционный анализ
Построение матрицы корреляций
Анализ мультиколлинеарности
Поиск нелинейных зависимостей и способов их устранения

5. Регрессионное моделирование (предсказание удоя)
Модель 1: базовая линейная регрессия
Модель 2: устранение нелинейностей (например, логика по СПО и ЭКЕ)
Модель 3: добавление новых признаков (например, имя отца)

Оценка моделей по:
R²
MSE, MAE, RMSE
Остаткам модели
Прогноз удоя для коров из cow_buy.csv

6. Классификация (вкус молока)
Построение модели логистической регрессии
Оценка по метрикам:
Accuracy
Recall (полнота)
Precision (точность)
Построение матрицы ошибок
Подбор порога классификации и минимизация ошибок второго рода
Прогноз вкуса молока у коров «ЭкоФермы»

7. Финальные рекомендации
Объединение прогнозов удоя и вкуса
Выбор коров, удовлетворяющих обоим условиям
Рекомендации фермеру:
Сколько коров можно купить без риска?
Какие улучшения можно внести в модели?
